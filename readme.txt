Connects Listagram.com list building optin popups with your Wordpress site so you can get conversion rates between 10-25% 

== Description ==
Listagram.com is a list building SaaS which performs fantastically on blogs and eCommerce sites. This module will connect your Listagram.com account with your Wordpress installation so you have full control of enabling optin forms directly from your Wordpress administration area.

Average conversion rate to newsletter using Listagram is currently 15% at 1,7 million views! This is the average amongst all our customers. Try us out today!

== Installation ==
Install like any other Wordpress plugin, navigate to \"Listagram\" admin menu item and enter your Listagram.com account token. Then you can enable it and start showing optin wheels on your Wordpress site.
